说明

  1. .write(true).append(true) 和 append(true) 是等价的
  2. create() 如果文件不存在，那么创建；否则，直接打开。为了文件可以打开，必须结合 write() 或 append() 方法一块使用
  3. create_new() 如果文件不存在，那么创建；否则，失败。此方法具有原子性，防止多个程序判断存在并一块创建。当create_new(true),那么.create()和truncate() 方法会自动忽略。此方法必须结合 write() 或 append() 方法一块使用
  4. File::create() 和 File::open() 也是这些方法的别名。

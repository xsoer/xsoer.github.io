PHP数组函数

发表于 2015-07-05 更新于 2020-04-18 分类于 PHP
  • 1.生成数组
    array()  []
  • 2.判断该值是否存在
    isset($arr[$i])
  • 3.判断是否是数组、
    is_array($arr)
  • 4.从尾部往数组内添加值
    array_push($arr,$a)
  • 5.从头部往数组内插入
    array_unshift
  • 6.弹出数组最后一个
    array_pop
  • 7.弹出数组第一个
    array_shift
  • 8.产生一个从0到6的数组
    range(0,6)
  • 9.检验是否在数组内
    in_array($a,$arr)
  • 10.返回数组的所有键值
    array_keys()
  • 11.改变键值为大写 或者CASE_LOWER 小写
    array_change_key_case($arr1,CASE_UPPER)
  • 12.合并两个或多个数组
    array_merge()
  • 13.给数组排序，升序
    sort($arr)
  • 14.查询数组内元素的个数
    count()
  • 15.用一个函数过滤数组
    array_filter($arr,$fun)
  • 16.输出数组
    print_r()
  • 17.遍历数组
    foreach($arr as $k)
  • 18.list遍历数组
    while(list($key,$value)==each($arr))
  • 19.去除字符串中的逗号，转为数组
    explode(',',$arr)
  • 20.将数组转为字符串
    implode(',',$arr)
  • 21.按键排序
    ksort()
  • 22.按值排序
    asort()
  • 23.降序排序
    rsort($arr)

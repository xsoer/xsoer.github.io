0x00、装饰器

  • 在函数调用前后想添加功能，但又不希望修改原函数的定义，这种在代码运行期间动态增加功能的方式，称之为“装饰器”（Decorator）。
  • 本质上，decorator就是一个返回函数的高阶函数。
  • 带了装饰器的函数会自动运行。
